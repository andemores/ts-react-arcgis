# note on Typescript

```
// Declare a tuple type
let x: [string, number];
// Initialize it
x = ["hello", 10]; // OK
// Initialize it incorrectly
x = [10, "hello"]; // Error
```

Consider   ```--strictNullChecks``` to avoid passing null/undefined when you shouldnt
